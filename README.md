# 🌊 Surf Run 2 — Beach Mission Arcade

**Surf Run 2** is a polished browser arcade runner where the player starts on the beach, completes missions, collects shells, avoids obstacles, and unlocks **Surf Mode** after gathering enough energy.

Live demo: **https://laurandreea10.github.io/Surf-run-2/**  
Repository: **https://github.com/LaurAndreea10/Surf-run-2**

---

## 🇬🇧 English

### Overview

Surf Run 2 continues the beach-runner idea with a more complete arcade structure: missions, modes, touch controls, keyboard controls, score, XP, local best score, sound toggle, reduced motion support, and a bilingual RO/EN interface.

The game is built with simple web technologies and can run directly from GitHub Pages.

### Features

- 🏖️ **Beach-to-water progression** — start on sand and unlock Surf Mode after collecting shells.
- 🎯 **Mission system** — collect shells, jump obstacles, and survive long enough.
- ⚡ **3 game modes**:
  - **Story** — balanced arcade experience.
  - **Rush** — faster and more intense.
  - **Zen** — relaxed mode without instant game over.
- 🌊 **Surf Mode** — new water visuals and obstacle types after unlocking.
- 🐚 **Collectibles** — shells increase score, XP, and mission progress.
- 🦀 **Obstacles** — crabs, umbrellas, sand castles, buoys, waves, and drift objects.
- 📱 **Mobile controls** — on-screen buttons and tap/swipe support.
- ⌨️ **Keyboard controls** — arrows, WASD, space, and pause.
- 🔊 **Sound toggle** — lightweight generated sound effects.
- ♿ **Accessibility options** — bilingual text, visible focus states, reduced motion toggle.
- 💾 **Local storage** — saves best score and language preference.

### Controls

| Action | Keyboard | Mobile |
|---|---|---|
| Move left | `←` / `A` | Left button / swipe left |
| Move right | `→` / `D` | Right button / swipe right |
| Jump | `Space` / `↑` / `W` | Jump button / tap canvas |
| Pause | `P` / Pause button | Pause button |

## 🇷🇴 Română

### Descriere

**Surf Run 2** este o continuare arcade pentru un joc de tip beach runner. Jucătorul începe pe plajă, completează misiuni, adună scoici, evită obstacole și deblochează **Surf Mode** după ce strânge suficientă energie.

Jocul este construit cu HTML, CSS și JavaScript simplu, fără framework-uri, și poate fi rulat direct prin GitHub Pages.

### Funcționalități

- 🏖️ **Progresie de la plajă spre apă** — începi pe nisip și deblochezi Surf Mode după 12 scoici.
- 🎯 **Sistem de misiuni** — adună scoici, sari peste obstacole și supraviețuiește 45 de secunde.
- ⚡ **3 moduri de joc**:
  - **Story** — experiență echilibrată.
  - **Rush** — mai rapid și mai intens.
  - **Zen** — mod relaxat, fără game over instant.
- 🌊 **Surf Mode** — vizualuri noi și obstacole de apă.
- 🐚 **Collectibles** — scoicile cresc scorul, XP-ul și progresul misiunilor.
- 🦀 **Obstacole** — crabi, umbrele, castele de nisip, balize, valuri și obiecte plutitoare.
- 📱 **Controale mobile** — butoane pe ecran, tap și swipe.
- ⌨️ **Controale tastatură** — săgeți, WASD, Space și Pause.
- 🔊 **Sunet opțional** — efecte audio generate direct în browser.
- ♿ **Accesibilitate** — interfață RO/EN, focus vizibil, reduced motion.
- 💾 **Salvare locală** — păstrează recordul și limba aleasă.

### Controale

| Săritură | `Space` / `↑` / `W` | Buton jump / tap pe canvas |
